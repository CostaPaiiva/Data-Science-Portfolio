# Machine Learning

Colecao de notebooks basicos de Machine Learning.

## Projetos

| Pasta | Tema | Arquivo principal |
| --- | --- | --- |
| `1-Ecomerce_Regressao_linear` | Regressao linear para dados de ecommerce | `Projeto_Ecomerce.ipynb` |
| `2-Regressao_logistica_Projeto` | Regressao logistica para anuncios | `Projeto_Regressao_Logistica.ipynb` |
| `3-Classificacao_Sentimentos_Naive Bayes` | Sentimentos com Naive Bayes | `Nayve_Bayes.ipynb` |
| `4-SVM_classificador_musicas` | Classificacao com SVM | `Machine Learning-SVM.ipynb` |
| `5-Clusterizacao_Flores_Iris_K-Means` | Clusterizacao com K-Means | `K-Means Iris.ipynb` |
| `6-Naive_Bayes_Analise_Sentimentos` | Analise de sentimentos com Naive Bayes | `Nayve_Bayes.ipynb` |
| `7-Arvore_decisao_flores_iris` | Arvore de decisao na base Iris | `Arvore-decisao.ipynb` |

## Como executar

Abra os notebooks no Jupyter, JupyterLab ou VS Code e execute as celulas em ordem.

Dependencias comuns:

```bash
python -m pip install pandas numpy scikit-learn matplotlib seaborn jupyter
```

Alguns notebooks de NLP podem precisar tambem de:

```bash
python -m pip install nltk
```

## Como trabalhar nos projetos

Entre na pasta de cada projeto, crie um ambiente virtual e instale as dependencias locais:

```bash
python -m venv .venv
.venv\Scripts\activate
python -m pip install -U pip
python -m pip install -r requirements.txt
```

Alguns notebooks nao possuem `requirements.txt`; nesses casos, consulte o README da pasta do notebook.
